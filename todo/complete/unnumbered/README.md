# Завершённые фичи без индекса (старый формат)

Сюда переносятся **реализованные** фичи, которые изначально были заведены в формате `FEATURE_NAME/` (без префикса `NNN-`).

Фичи с нумерацией (`001-LOCALIZATION`, `002-…`) после завершения остаются в корне `todo/complete/` с тем же именем, например `todo/complete/001-LOCALIZATION/`.

## Как переносить

Когда фича из `todo/WIREGUARD_URI/`, `todo/OUTBOUNDS_CONFIGURATOR/` и т.п. реализована и отчёт заполнен:

1. Переместить всю папку фичи из `todo/` в `todo/complete/unnumbered/`.
2. При необходимости обновить ссылки в `todo/README.md` (секция «Сделанное») и в этом README.

## Текущее содержимое

Папка пуста. Ранее завершённые фичи получили номера 001–010 по истории git и находятся в корне `todo/complete/` (001-FEATURES_2025, 002-WIZARD_STATE, … 010-OUTBOUND_EDIT_PREVIEW_TAB). Сюда переносят только новые завершённые фичи без индекса.
